This page is the index of every decorator the framework ships, grouped
by concern. Skim the table for the one you need, click through to the
page that explains it in context. Each row says where the decorator
goes (struct, impl block, method, field) so you can confirm the
position without leaving the page.

## DI and lifecycle

| Decorator | Goes on | Summary | Read more |
|---|---|---|---|
| `#[module]` | struct | Composes a binary: lists `providers` it owns and `imports` it depends on | [Modules](/fundamentals/modules/) |
| `#[injectable]` | struct | Marks a provider built from the container, with optional `scope = request \| transient` | [Providers](/fundamentals/providers/) |
| `#[inject]` | field | Resolves the field from the container at construction | [Providers](/fundamentals/providers/) |
| `#[hooks]` | impl block | Orchestrates lifecycle methods on a provider | [Lifecycle](/fundamentals/lifecycle/) |
| `#[on_module_init]` | method | Runs once after wiring, before serving | [Lifecycle](/fundamentals/lifecycle/) |
| `#[on_application_bootstrap]` | method | Runs once after every module init, before serving | [Lifecycle](/fundamentals/lifecycle/) |
| `#[on_module_destroy]` | method | Runs on shutdown after transports stop | [Lifecycle](/fundamentals/lifecycle/) |
| `#[before_application_shutdown]` | method | Runs before the application begins shutting down | [Lifecycle](/fundamentals/lifecycle/) |
| `#[on_application_shutdown]` | method | Runs last during shutdown, best-effort | [Lifecycle](/fundamentals/lifecycle/) |

## HTTP

| Decorator | Goes on | Summary | Read more |
|---|---|---|---|
| `#[controller]` | struct | Mounts the struct as a controller with a base path and optional `version` | [Controllers](/http/controllers/) |
| `#[routes]` | impl block | Wires methods to HTTP routes and emits the route table | [Controllers](/http/controllers/) |
| `#[get]` `#[post]` `#[put]` `#[patch]` `#[delete]` | method | Binds the method to an HTTP verb at the given path | [Controllers](/http/controllers/) |
| `#[http_code]` | method | Overrides the default response status code | [Responses](/http/responses/) |
| `#[response_header]` | method | Appends a response header (repeatable) | [Responses](/http/responses/) |
| `#[redirect]` | method | Issues a redirect and discards the handler payload | [Responses](/http/responses/) |
| `#[use_guards]` | struct or method | Binds container-resolved Guards, first listed outermost | [Guards](/fundamentals/guards/) |
| `#[use_interceptors]` | struct or method | Binds container-resolved Interceptors | [Interceptors](/fundamentals/interceptors/) |
| `#[use_filters]` | struct or method | Binds exception Filters, wrapping outside the guards | [Exception filters](/fundamentals/exception-filters/) |
| `#[meta]` | method | Attaches typed metadata read back via `Reflector` (repeatable) | [Guards](/fundamentals/guards/#attach-context-and-route-metadata) |
| `#[public]` | method | Marks the route public so auth guards skip it | [Guards](/fundamentals/guards/#attach-context-and-route-metadata) |
| `#[interceptor]` | struct | Marks a struct as a global HTTP interceptor | [Interceptors](/fundamentals/interceptors/) |
| `#[input]` | struct | Shorthand for input DTOs (`Deserialize` + `Validate` + `deny_unknown_fields`) | [Extractors](/http/extractors/) |

## GraphQL

| Decorator | Goes on | Summary | Read more |
|---|---|---|---|
| `#[resolver]` | struct + impl block | Marks a resolver provider and orchestrates its operations | [GraphQL](/graphql/) |
| `#[query]` | method | Root query operation; requires an access posture (`#[authorize]` or `#[public]`) | [Queries and mutations](/graphql/queries-mutations/) |
| `#[mutation]` | method | Root mutation operation; requires an access posture (`#[authorize]` or `#[public]`) | [Queries and mutations](/graphql/queries-mutations/) |
| `#[authorize]` | method | Access posture: gates the operation by ability (`Action`, `Entity`) and masks the response | [Queries and mutations](/graphql/queries-mutations/) |
| `#[public]` | method | Access posture: deliberately ungated operation — no gate, no mask | [Queries and mutations](/graphql/queries-mutations/) |
| `#[field_resolver]` | method | Custom field resolver on a wire type; takes no posture attribute | [Field resolvers](/graphql/field-resolvers/) |
| `#[dataloader]` | impl block on service | Batches a method into a request-scoped DataLoader | [Dataloader](/graphql/dataloader/) |
