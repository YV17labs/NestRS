## Prerequisites

NestRS needs **Rust 1.95+** (edition 2024). Install [rustup](https://rustup.rs)
if you don't have it:

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Then the CLI, plus [`just`](https://github.com/casey/just) for the recipes the
scaffolder writes:

```bash
cargo install --locked nest-rs-cli just
```

## Scaffold and start

Pick a layout — a **workspace** (monorepo: shared `crates/features/`, one binary
per app under `apps/`) or a **standalone** crate (everything in `src/`).

<Tabs syncKey="getting-started-cli">
<TabItem label="Workspace">

<Steps>

1. Scaffold.

   ```bash
   nestrs new acme
   cd acme
   ```

2. Start the default app.

   ```bash
   just dev hello
   ```

   Open **http://localhost:3000/** — `Hello World`, served from
   `crates/features/hello/`. `apps/hello/` only wires modules.

</Steps>

</TabItem>
<TabItem label="Standalone">

<Steps>

1. Scaffold.

   ```bash
   nestrs new acme --standalone
   cd acme
   ```

2. Start it.

   ```bash
   just dev
   ```

   Open **http://localhost:3000/** — `Hello World`, from `src/`.

</Steps>

</TabItem>
</Tabs>

That's the loop: scaffold, `just dev`, open the browser. Grow the project with
`nestrs g feature|resource|<transport>` — see the [CLI reference](/cli/).
